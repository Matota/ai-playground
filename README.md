# AI Playground

Welcome to my AI Playground! This repository documents my journey learning Artificial Intelligence, starting from basic NLP to Computer Vision and Generative AI.

## Projects

### 1. Sentiment Analysis (NLP)
A simple sentiment analysis model built with Python and Scikit-learn.
- **Goal**: Classify text as positive or negative.
- **Technique**: TF-IDF Vectorization + Logistic Regression.
- **File**: `sentiment_analysis.py`

#### How to Run
1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run the script:
    ```bash
    python3 sentiment_analysis.py
    ```
